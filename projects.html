Software Development Projects

fsr package

    As a result of my final monography of my Data Science Specialization Course, I developed together with my supervisor, an R package to work with Fuzzy Logic and Spatial Data. The fsr package transform spatial data into fuzzy spatial data.

    It is an open source project and published on CRAN (Comprehensive R Archive Network), the official package repository for R packages.

    My participation in this project was as the main developer, always with the support of my supervisor, which also has coded but more important, was the main source of the Fuzzy concepts and all types of manipulation implemented.

    Main knowledge

    • Software Development

    • Documentation
    • Scikit-learn
